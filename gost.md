# Правила форматирования (ГОСТ РФ)

1. АБЗАЦЫ И ИНТЕРВАЛЫ (ВАЖНО): Строго запрещено делать пустые строки (отбивки) между абзацами. Каждый новый абзац должен начинаться с новой строки, но БЕЗ пропуска пустой строки перед ним. Текст должен идти сплошным потоком, чтобы его было удобно копировать в Microsoft Word.
2. КАВЫЧКИ: Используй только «елочки». Для кавычек внутри кавычек — „лапки“.
3. ТИРЕ: Не использовать длинное тире.
4. СОКРАЩЕНИЯ: Общепринятые сокращения пиши корректно (т. е., т. д., т. п., др. — с пробелом, если это несколько слов).
5. ССЫЛКИ: Внутритекстовые ссылки оформляй строго в квадратных скобках: [1, с. 25] или [3].
